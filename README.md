# XRPL Consensus Protocol

This repository specifies the XRP Ledger consensus protocol.

- **[The paper](paper/main.pdf)** ([`paper/`](paper/)): a formal specification with a model, construction, and analysis, abstracting away engineering details.
- **[The docs](docs/)** ([`docs/README.md`](docs/README.md)): a documentation of the protocol as `xrpld` implements it.
